# Contributing

Thanks for your interest in contributing! Please follow these simple steps:

1. Fork the repository.
2. Create a branch for your change: `git checkout -b feature/my-change`.
3. Make your changes with clear, focused commits.
4. Run notebooks locally and ensure examples run as expected.
5. Open a pull request describing your change and why it's useful.

Be respectful and follow common sense: write clear commits, keep changes small, and include context in PR descriptions.
