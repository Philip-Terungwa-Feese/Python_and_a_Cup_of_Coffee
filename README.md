# Python Programming Basics for Beginners

Clear, practical Jupyter notebooks and resources to learn Python from first principles.

## Overview

This repository contains interactive notebooks and teaching materials designed for absolute beginners. The content walks learners through fundamental Python concepts with examples, exercises, and short projects.

## What You'll Learn

- Variables and data types
- Operators and expressions
- Control flow (if / elif / else)
- Loops (`for`, `while`) and iteration patterns
- Functions and code organization
- Lists, dictionaries, and common data structures
- Debugging tips and best practices

## Quickstart

Prerequisites: Python 3.8+ and either Jupyter Notebook or JupyterLab.

1. Clone the repository:

```bash
git clone https://github.com/Philip-Terungwa-Feese/Python_and_a_Cup_of_Coffee.git
cd Python_and_a_Cup_of_Coffee
```

2. Create and activate the project virtual environment:

```bash
python -m venv .venv
# Windows PowerShell
.venv\Scripts\Activate.ps1
# Windows CMD
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate
```

3. Install the project dependencies and start the notebooks:

```bash
pip install --upgrade pip
pip install -r requirements.txt
jupyter notebook
```

Open `Python_Basics_for_Beginners.ipynb` in the browser and follow the sections.

## Get Started

Once the virtual environment is active, run:

```bash
pip install -r requirements.txt
jupyter notebook
```

Then open `Python_Basics_for_Beginners.ipynb` and select the `Python (.venv)` kernel if needed.
